GITHUB PAGES DEPLOYMENT COMMANDS - COPY & PASTE

=== STEP 1: Install Dependencies ===
cd frontend
npm install

=== STEP 2: Update Homepage URL ===
Replace "yourusername" in package.json with your actual GitHub username

=== STEP 3: Push to GitHub ===
cd ..
git add .
git commit -m "Frontend ready for GitHub Pages deployment"
git push origin main

=== STEP 4: Deploy to GitHub Pages ===
cd frontend
npm run deploy

=== STEP 5: Enable GitHub Pages ===
1. Go to your GitHub repository
2. Click "Settings" tab
3. Scroll to "Pages" section
4. Source: "Deploy from a branch"
5. Branch: "gh-pages"
6. Click "Save"

=== YOUR FRONTEND URL WILL BE ===
https://yourusername.github.io/loan-crm

Replace "yourusername" with your actual GitHub username!
